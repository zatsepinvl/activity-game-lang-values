    0. Для игры Вам необходимо: не менее 4 игроков и хорошее настроение =).
    1. Все участники делятся на команды по 2 человека минимум в каждой. Задача команд - набрать наибольшее количество баллов путем отгадывания слов. Необходимое количество баллов для победы оговаривается перед началом игры.
    2. Каждая команда по очереди начинает раунд. Цвет подскажет какая команда играет в данный момент.
    3. Один из игроков команды - ведущий, он должен устно объяснить, как можно больше слов остальным членам команды за определенное время с момента включения таймера. После каждого раунда ведущий в команде меняется по очереди. Разрешается пропустить слово за штраф (-1 балл). Разрешается использовать только действие «Рассказать».
        3.1. Рассказать - ведущий должен объяснить слово используя устную форму общения. Запрещается употреблять производные или однокоренные слова с загаданным словом.
    4. После истечения времени команда должна честно отметить отгаданные слова. После чего очередь переходит к следующей команде.
    5. Игра заканчивается, когда одна из команд набирает нужное количество баллов. При равном количестве баллов (необходимых для победы) у нескольких команд, игра ведется до тех пор, пока одна из команд не поведет в счете.
        5.1 Лидирующей командой считается та, которая имеет максимальное количество очков по окончанию раунда.
